def evaluation_grid (   self)

The nodes sorted into a 2D grid based on their dependency.

Rows affect each other and have to be evaluated in sequence. The Nodes on each row however can be evaluated in parallel as they are independent of each other. The amount of Nodes in each row can vary. Returns: (list of list of INode): Each sub list represents a row.
